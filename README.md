# card-checking-

! ВНИМАНИЕ !
ВСЯ работа происходила в Visual Studio Code.
Чтобы использовать мой репозщиторий вы можете просто скачатль zip архив, либо использовать 
в консоли в вашей папке - git clone https://github.com/shinadsu/card-checking-.git - 

1. Сервер или расширение. 
Первое и самое необходимое, что нам надо сделать так это поставить наш сервер
в качестве своего я использовал OsPanel. Если его нет, то можно поставить Live Server, {ritwickdey.LiveServer} вот его id VsCode

2. Настройка путей. 
Самое необходимое, что нам нужно сделать это прописать свои пути в файле /app/src/core.php на строчке 8 
в ковычках нужно указать свою путь куда вы установили репозиторий, проверьте чтобы путь был правильный иначе будет ошибка

3. Подключение к api. 
Подключение к api уже полностью настроено, вам не нужно ничего делать/менять

4. Работа с файлом index.php
Основная работа всего файла заклчюается в php коде, который соверашет все необходимое. 
В файле указаны необходимые коментарии 

5. Файл config.json. 
Файл config.json содержит в себе логин и пароль для доступа к api.
